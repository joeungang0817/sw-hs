프로젝트 소개

이 프로젝트는 웹 프론트엔드의 핵심 개념을 실습하기 위해 만들어졌습니다.

학습 목표

HTML

시맨틱 태그를 사용하여 검색엔진과 스크린리더가 이해할 수 있는 구조적인 문서를 작성하는 법을 배웁니다.

CSS

Cascading 원리, Box Model, Flexbox, CSS 변수, 반응형 디자인, 애니메이션까지 핵심 스타일링 기법을 익힙니다.

JavaScript

변수, 배열, 객체부터 클래스, 상속, OOP 4대 원칙까지. DOM 조작과 이벤트 처리로 동적인 웹 페이지를 만듭니다.

실무 스킬

다크모드 토글, 햄버거 메뉴, 스크롤 페이드인 등 실제 웹사이트에서 자주 쓰이는 기능을 직접 구현합니다.

사용된 핵심 개념 목록

  • 시맨틱 HTML: header, nav, main, section, article, footer
  • CSS Cascading: 외부 < 내부 < 인라인 우선순위
  • CSS Box Model: Block, Inline, Inline-Block 차이
  • CSS 변수: :root에서 색상/크기를 한곳에서 관리
  • Flexbox: justify-content, align-items로 정렬
  • 반응형: @media 쿼리로 모바일 대응
  • 애니메이션: opacity, transform, transition으로 페이드인
  • JS 기초: const, Array, Object, 함수, 조건문, 반복문
  • JS OOP: Class, constructor, extends, super
  • OOP 4원칙: 캡슐화, 상속, 다형성, 추상화
  • DOM 조작: querySelector, addEventListener
  • 실무: 다크모드, 햄버거 메뉴, IntersectionObserver
