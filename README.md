## 💡 1주차 핵심 요약
[cite_start]첫 번째 시간에는 웹 서비스의 3가지 핵심 요소와 협업을 위한 기초 도구를 배웠습니다[cite: 58, 127].

1. **프론트엔드 (Frontend)**
   - [cite_start]HTML(뼈대), CSS(옷), React(뇌)를 이용해 사용자의 눈에 보이는 UI와 상호작용을 만듭니다[cite: 64, 65, 68, 70, 78].
2. **백엔드 (Backend)**
   - [cite_start]Node.js와 Express를 사용하며, 프론트엔드의 요청을 받아 데이터를 처리하고 검증하는 보이지 않는 주방장 역할을 합니다[cite: 85, 87, 89, 90, 94].
3. **데이터베이스 (Database)**
   - [cite_start]MySQL을 활용해 회원 정보, 게시글 등 서비스의 모든 데이터를 체계적으로 저장하고 관리하는 창고입니다[cite: 97, 98, 99, 101, 102].
4. **버전 관리와 환경 세팅**
   - [cite_start]**WSL:** 윈도우 환경에서 가볍고 빠르게 리눅스 개발 환경을 구축합니다[cite: 177, 178, 181].
   - [cite_start]**Git & GitHub:** 코드의 버전을 관리하고 팀원들과 안전하게 협업(Clone, Commit, Push, Pull, PR)하기 위해 필수적으로 사용합니다[cite: 127, 128, 130, 140, 144, 146, 148].
