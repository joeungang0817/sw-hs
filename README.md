## 💡 1주차 요약: 웹 서비스는 어떻게 굴러갈까?
첫 시간에는 앞으로 우리가 만들 웹 서비스의 3가지 핵심 역할과, 코딩을 위한 기본 도구들을 세팅해 봤어요.

1. **프론트엔드 (Frontend)**
   - 인스타그램 화면처럼 사용자 눈에 보이는 모든 것! HTML(뼈대), CSS(옷), JS/React(뇌)를 이용해 만듭니다.
2. **백엔드 (Backend)**
   - 식당의 주방장처럼, 뒤에서 데이터와 요청을 처리하는 역할을 해요. 우리는 Node.js와 Express를 사용합니다.
3. **데이터베이스 (Database)**
   - 회원정보나 게시글을 체계적으로 저장해 두는 거대한 창고예요. (MySQL 사용)
4. **버전 관리와 세팅 (WSL & Git)**
   - 본격적인 개발을 위해 가벼운 리눅스 환경(WSL)을 깔고, 팀원들과 코드를 안전하게 공유하기 위해 Git과 GitHub 사용법을 익혔습니다. (명령어 잊어버리면 언제든 다시 찾아보세요!)
